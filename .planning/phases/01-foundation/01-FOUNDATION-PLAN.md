# Milestone 1: Saphir Foundation (Wave 1)

## Goal
Establish the core technical foundation for the Saphir theme, ensuring high performance, Elementor compatibility, and adherence to the luxury hotel design aesthetic.

## Deliverables
- [ ] Implement fluid typography using `clamp()` logic.
- [ ] Integrate Accessibility Guardrails (automated checks).
- [ ] Build standalone Booking Sticky Bar (Custom Elementor Widget).
- [ ] Build "Concierge-Style" Predictive Search (Custom Elementor Widget).
- [ ] Implement asset-budgeting strategy in `functions.php`.

## Acceptance Criteria
- Fluid typography scales smoothly without breaking layout.
- Booking bar and Search components are functional via custom Elementor Widgets, offering a native drag-and-drop experience for editors.
- Elementor's default asset overhead is reduced for critical high-performance pages.
- Accessibility meets a minimum 4.5:1 contrast ratio for primary text.
